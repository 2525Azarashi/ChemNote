# 表記ガイド（UNIT_GUIDE）

ChemNote の問題文・解説・Step解説における化学式・単位・数式の**表記統一ルール**です。
新しい問題やStep解説を追加・編集するときは、必ずこのガイドに従ってください。
機械的なチェックは `npm run lint:chem`（lint-chem）で行えます。

---

## 1. 基本方針

- **数値・単位・数式に使う英数字・記号は半角に統一する。**
  PDF から抽出した問題文には全角の英数字（`ｍｏｌ`, `０．５０`, `Ｌ` など）が混ざっているため、
  これらは半角（`mol`, `0.50`, `L`）へ正規化する。
- **日本語の文章として自然な全角約物（`（）`『：』全角空白など）はそのまま残す。**
  過剰な半角化で日本語の可読性を損なわないことを優先する。

---

## 2. 数値

| 項目 | ルール | 良い例 | 悪い例 |
|------|--------|--------|--------|
| 数字 | 半角 | `0.50`, `1.0×10⁻¹⁴` | `０．５０`, `１．０` |
| 小数点 | 半角ピリオド `.` | `0.010` | `０．０１０`, `0．010` |
| 桁区切り | 使わない | `1000` | `1,000` |
| 指数 | 上付き文字（`⁻`, `¹⁴` など）を使用 | `1.0×10⁻⁷` | `1.0*10^-7` |

## 3. 単位

| 単位 | 表記 | 備考 |
|------|------|------|
| 物質量 | `mol` | 全角 `ｍｏｌ` は不可 |
| モル濃度 | `mol/L` | `mol・L⁻¹` / 全角 `ｍｏｌ／Ｌ` は使わず `mol/L` に統一 |
| 体積 | `L`, `mL` | 全角 `Ｌ`, `ｍＬ` は不可 |
| 質量 | `g`, `kg` | |
| 温度 | `℃` | 単位記号としての `℃`（U+2103）を使用。`°C` は使わない |
| 圧力 | `Pa`, `hPa` | |

## 4. 数式・演算子

| 記号 | 表記 | 備考 |
|------|------|------|
| 乗算 | `×`（U+00D7） | ASCII の `x` や `*` は使わない。全角 `×` はそのまま可（正しい乗算記号） |
| 除算（単位以外） | `÷` または分数 | |
| 等号 | 半角 `=` | 全角 `＝` は不可 |
| 加算・減算 | 半角 `+`, `-` | 全角 `＋` は不可。マイナスは指数では上付き `⁻` |
| 不等号 | 半角 `<`, `>`, `≦`, `≧` | 全角 `＜`, `＞` は不可 |
| パーセント | 半角 `%` | 全角 `％` は不可 |
| スラッシュ | 半角 `/` | 全角 `／` は不可 |

## 5. 化学式・イオン

| 項目 | ルール | 良い例 |
|------|--------|--------|
| 元素記号 | 半角、正しい大文字小文字 | `Na`, `Cl`, `H₂O` |
| 添え字（原子数） | 下付き文字 | `H₂SO₄`, `CO₃²⁻` |
| 電荷 | 上付き文字 | `H⁺`, `OH⁻`, `Ca²⁺` |
| モル濃度の表現 | 半角角括弧 `[ ]` | `[H⁺]`, `[OH⁻]` |
| pH / Kw 等 | 半角 | `pH`, `Kw`, `Ka` |

## 6. 句読点・約物（そのまま残すもの）

- 日本語文中の丸括弧は全角 `（ ）` を使用（例：`（１価の強酸）` は数字のみ半角化して `（1価の強酸）`）。
- 全角コロン `：`、全角空白 `　`、読点 `、`・句点 `。` はそのまま。
- 中点 `・`（列挙・化合物の水和など）はそのまま。

---

## 7. lint-chem の使い方

```bash
# 表記ゆれを検出（CIやコミット前に実行）
npm run lint:chem

# 検出した表記ゆれを自動修正（安全な正規化のみ適用）
npm run lint:chem:fix
```

- `lint:chem` は違反箇所（ファイル・行・内容）を一覧表示し、違反があれば終了コード 1 を返す。
- `lint:chem:fix` は本ガイドの「安全に正規化できる」ルール（全角英数字・全角スラッシュ・全角小数点・
  全角角括弧・全角演算子 `＝＋＜＞％`）のみを半角へ変換する。日本語約物や `×` `÷` `：` は変更しない。
- 自動修正後は必ず `npm run lint`（tsc）と `npm run build` で壊れていないか確認すること。
