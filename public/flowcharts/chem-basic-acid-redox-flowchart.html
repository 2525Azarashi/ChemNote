第2部 物質の変化 ／ 2、酸と塩基

重要事項① 〜酸と塩基の定義（アレニウス／ブレンステッド・ローリー）〜

酸と塩基をどう定義するか？▶
【Step1：アレニウスの定義（水溶液限定）】
電離して H⁺ を生じる物質
ⓘ
アレニウスの酸＝水に溶かして電離したとき、H⁺（＝オキソニウムイオン H₃O⁺）を生じる物質。
例①：HCl → H⁺ + Cl⁻（塩酸）
例②：H₂SO₄ → 2H⁺ + SO₄²⁻（硫酸）
例③：HNO₃ → H⁺ + NO₃⁻（硝酸）
例④：CH₃COOH ⇄ CH₃COO⁻ + H⁺（酢酸／可逆）
厳密には：H⁺ は水分子と結びついて オキソニウムイオン H₃O⁺ として存在する。
例：HCl + H₂O → H₃O⁺ + Cl⁻
電離して OH⁻ を生じる物質
ⓘ
アレニウスの塩基＝水に溶かして電離したとき、水酸化物イオン OH⁻を生じる物質。
例①：NaOH → Na⁺ + OH⁻
例②：KOH → K⁺ + OH⁻
例③：Ca(OH)₂ → Ca²⁺ + 2OH⁻
例④：Ba(OH)₂ → Ba²⁺ + 2OH⁻
欠点：アンモニア NH₃ は OH⁻ を式中に含まないのに塩基性を示す → アレニウスの定義では説明できない！
→ そこで登場したのが「ブレンステッド・ローリーの定義」。
【Step2：ブレンステッド・ローリーの定義（H⁺のやり取り）】
H⁺ を「与える」物質
ⓘ
ブレンステッド・ローリーの酸＝反応においてH⁺（プロトン）を相手に与える物質。
例①：HCl + H₂O → H₃O⁺ + Cl⁻
　　　　→ HCl が H⁺ を H₂O に与えた → HCl が酸／H₂O が塩基
ポイント：水溶液以外の反応にも定義が使える！（気相・非水溶媒での反応もOK）
H⁺ を「受け取る」物質
ⓘ
ブレンステッド・ローリーの塩基＝反応においてH⁺ を相手から受け取る物質。
例②：NH₃ + H₂O ⇄ NH₄⁺ + OH⁻
　　　　→ H₂O が H⁺ を NH₃ に与えた → H₂O が酸／NH₃ が塩基
ここで注目！：同じ H₂O が、相手によって酸にも塩基にもなる！
これを 両性物質（両性化合物） と呼ぶ。
【Step3：共役酸・共役塩基（発展）】
H⁺のやり取りペア
ⓘ
ブレンステッドの定義では、逆反応で H⁺ をやり取りするペアを共役酸・共役塩基とよぶ。
NH₃ + H₂O ⇄ NH₄⁺ + OH⁻
　塩基　酸　　共役酸　共役塩基

・NH₃ の共役酸＝NH₄⁺（H⁺を1個受け取った形）
・H₂O の共役塩基＝OH⁻（H⁺を1個失った形）
💡 コツ：「H⁺を1個外した／1個くっつけた形」を作れば共役ペアの完成！
【Step4：確認・練習】
酸／塩基を答える
ⓘ
次の反応で、ブレンステッド・ローリーの酸・塩基はそれぞれ何か？
❶ HCl + NH₃ → NH₄Cl
　　→ 酸＝HCl（H⁺を与える）／塩基＝NH₃（H⁺を受け取る）

❷ CO₃²⁻ + H₂O ⇄ HCO₃⁻ + OH⁻
　　→ 酸＝H₂O／塩基＝CO₃²⁻

❸ HSO₄⁻ + H₂O ⇄ SO₄²⁻ + H₃O⁺
　　→ 酸＝HSO₄⁻／塩基＝H₂O

重要事項② 〜酸と塩基の強さ（電離度・価数）〜

強い？弱い？何価？▶
【Step1：電離度αとは】
電離した割合を示す値
ⓘ
電離度 α＝電解質が水に溶けたとき、溶解した全体のうち電離した割合。
電離度 α = 電離した電解質の物質量 ÷ 溶解した全体の物質量
（0 ＜ α ≦ 1）
💡 濃度が小さい（薄い）ほど、電離度 α は大きくなる。
例題：酢酸 0.10 mol を水に溶かすと H⁺ が 0.0010 mol 生じた。電離度は？
解答：α = 0.0010 ÷ 0.10 = 0.010
【Step2：強酸・弱酸／強塩基・弱塩基】
強酸は3つだけ！
ⓘ
強酸＝水溶液中でほぼ完全に電離（α ≒ 1）
★ 強酸は3つだけ ★
塩酸 HCl／硫酸 H₂SO₄／硝酸 HNO₃　と 3つ唱える！
弱酸＝電離度 α が小さい酸（それ以外はほぼ弱酸）
弱酸の代表：HF（フッ化水素）、CH₃COOH（酢酸）、H₂S（硫化水素）、(COOH)₂（シュウ酸）、H₃PO₄（リン酸）、H₂CO₃（炭酸）
1族＋Mg以外の2族＝強塩基
ⓘ
強塩基＝水溶液中でほぼ完全に電離（α ≒ 1）
★ 強塩基の覚え方 ★　1族 と（Mgを除く）2族 の水酸化物

・NaOH（水酸化ナトリウム）／KOH（水酸化カリウム）＝1族
・Ca(OH)₂（水酸化カルシウム）／Ba(OH)₂（水酸化バリウム）＝2族
弱塩基＝電離度が小さい塩基
弱塩基の代表：NH₃（アンモニア）、Mg(OH)₂、Cu(OH)₂、Al(OH)₃、Fe(OH)₃
【Step3：価数】
出せる H⁺ の数
ⓘ
酸の価数＝電離して H⁺ になれるH の数。
価数強酸弱酸
1価HCl、HNO₃HF、HBr、HI、CH₃COOH
2価H₂SO₄H₂S、(COOH)₂（シュウ酸）、H₂CO₃
3価―H₃PO₄（リン酸）
注意：酢酸 CH₃COOH の H は -COOH の H だけが電離して1価。（メチル基-CH₃のHは電離しない）
出せる OH⁻ の数
ⓘ
塩基の価数＝電離して OH⁻ になれるOH の数（NH₃ は「H⁺を受け取る数」で1価）。
価数強塩基弱塩基
1価NaOH、KOHNH₃
2価Ca(OH)₂、Ba(OH)₂Mg(OH)₂、Cu(OH)₂、Fe(OH)₂
3価―Al(OH)₃、Fe(OH)₃
【Step4：★超重要★ 分類まとめ表】
テスト直前に見返す表
ⓘ
強酸（α≒1）弱酸（α小）強塩基（α≒1）弱塩基（α小）
1価HCl／HNO₃HF／CH₃COOH／HBr／HINaOH／KOHNH₃
2価H₂SO₄H₂S／(COOH)₂／H₂CO₃Ca(OH)₂／Ba(OH)₂Mg(OH)₂／Cu(OH)₂
3価―H₃PO₄―Al(OH)₃／Fe(OH)₃

重要事項③ 〜pH について（水のイオン積・計算）〜

酸性・中性・塩基性を数値化！▶
【Step1：pHの定義】
[H⁺] の指数を取り出した値
ⓘ
pH の定義：水素イオン濃度 [H⁺] を用いて、
[H⁺] = 1.0 × 10⁻ˣ mol/L　のとき　pH = x

（正式には pH = -log₁₀[H⁺] だが、化学基礎では指数の x を読むだけで OK！）
🌡 pH の目安
pH ＜ 7 → 酸性（数値が小さいほど強酸性）
pH ＝ 7 → 中性
pH ＞ 7 → 塩基性（数値が大きいほど強塩基性）
【Step2：水のイオン積 Kw】
[H⁺]×[OH⁻]は一定！
ⓘ
純水の中でも水はごくわずかに電離している：H₂O ⇄ H⁺ + OH⁻
【水のイオン積】25℃で一定
Kw = [H⁺][OH⁻] = 1.0 × 10⁻¹⁴ (mol/L)²

→ 指数どうしの関係：[H⁺]の指数 + [OH⁻]の指数 = -14

中性のとき：[H⁺] ＝ [OH⁻] ＝ 1.0 × 10⁻⁷ mol/L
💡 使い方：[H⁺] が分かれば [OH⁻] を、[OH⁻] が分かれば [H⁺] を計算できる！
【Step3：pH の計算パターン】
α＝1として扱う
ⓘ
強酸は α = 1として計算する。
例題1：0.010 mol/L の HCl 水溶液の pH は？
HCl は1価の強酸なので → [H⁺] = 0.010 = 1.0 × 10⁻² mol/L
∴ pH = 2
例題2：0.050 mol/L の H₂SO₄ 水溶液の pH は？
H₂SO₄ は2価の強酸なので → [H⁺] = 0.050 × 2 = 0.10 = 1.0 × 10⁻¹ mol/L
∴ pH = 1
濃度×価数×αを使う
ⓘ
弱酸は電離度 α をかける
[H⁺] = 酸のモル濃度 × 価数 × 電離度 α
例題：0.10 mol/L の酢酸（α = 0.010）の pH は？
[H⁺] = 0.10 × 1 × 0.010 = 1.0 × 10⁻³ mol/L
∴ pH = 3
Kwを利用して[H⁺]を求める
ⓘ
塩基は [OH⁻] を求めた後、水のイオン積 を使って [H⁺] に変換する。
例題：0.010 mol/L の NaOH 水溶液の pH は？
NaOH は1価の強塩基 → [OH⁻] = 0.010 = 1.0 × 10⁻² mol/L
水のイオン積より：[H⁺] × 1.0 × 10⁻² = 1.0 × 10⁻¹⁴
∴ [H⁺] = 1.0 × 10⁻¹² mol/L → pH = 12

裏ワザ：[OH⁻] = 10⁻² のとき、指数の和 = -14 より [H⁺] の指数 = -12 → pH = 12
【Step4：希釈と pH】
薄めても pH は 7 を越えない
ⓘ
酸を10倍に薄めると：[H⁺] が 1/10 → pH は 1 増える
塩基を10倍に薄めると：[OH⁻] が 1/10 → pH は 1 減る

⚠ 注意：薄めすぎても pH は「7」を超えることはない！
（薄い酸でも塩基性にはならない。水そのものの H⁺ が効いてくるため）
【Step5：pH指示薬・pH測定】
MO・PP・BTB
ⓘ
指示薬変色域pH酸性側塩基性側覚え方
メチルオレンジ MOpH 3.1〜4.4赤黄ベーコンエッグ（赤→黄）
フェノールフタレイン PPpH 8.0〜9.8無色赤（桃）塩基性で赤くなる

重要事項④ 〜中和とは何か（塩の分類と液性）〜

中和 → 塩 → 液性を判定！▶
【Step1：中和反応とは】
酸の H⁺ + 塩基の OH⁻ → H₂O
ⓘ
中和＝酸の H⁺ と塩基の OH⁻ が結びついて、水（H₂O）と塩（えん）ができる反応。
本質の式：H⁺ + OH⁻ → H₂O（発熱反応 ＝ 中和熱）

例①：HCl + NaOH → NaCl + H₂O
例②：H₂SO₄ + 2NaOH → Na₂SO₄ + 2H₂O
例③：HCl + NH₃ → NH₄Cl（H₂Oが出ない中和もある！）
💡 中和で生じるイオン化合物を「塩（えん）」とよぶ。塩＝酸の陰イオン＋塩基の陽イオン。
【Step2：塩の分類（化学式で見分ける）】
式の中に H や OH が残るかで判別
ⓘ
分類特徴（化学式）例
正塩H も OH も含まないNaCl／Na₂SO₄／KNO₃／CH₃COONa／NH₄Cl／Na₂CO₃
酸性塩式に酸のHが残るNaHSO₄／NaHCO₃／KH₂PO₄
塩基性塩式に塩基のOHが残るMgCl(OH)／CuCl(OH)
⚠ 「酸性塩＝酸性」ではない！ 分類（式の見た目）と液性（水に溶かしたときの pH）は別モノ！
【Step3：液性の判定（超頻出！）】
元の「強い方が勝つ」
ⓘ
正塩の液性は「もとの酸と塩基の強さ」で決まる！
もとの酸もとの塩基液性例
強酸強塩基中性NaCl（HCl+NaOH）／Na₂SO₄／KNO₃
強酸弱塩基酸性NH₄Cl（HCl+NH₃）／CuSO₄
弱酸強塩基塩基性CH₃COONa（酢酸+NaOH）／Na₂CO₃
弱酸弱塩基ほぼ中性CH₃COONH₄
💡 覚え方：「強い方が勝つ！」 強酸 vs 弱塩基 → 酸性が勝つ／弱酸 vs 強塩基 → 塩基性が勝つ
NaHSO₄ vs NaHCO₃
ⓘ
酸性塩はもとの酸・塩基の強さで判断するのがルール。ただし例外あり。
❶ NaHSO₄（強酸 H₂SO₄ + 強塩基 NaOH の酸性塩）
　→ 電離して H⁺ を出す → 酸性！

❷ NaHCO₃（弱酸 H₂CO₃ + 強塩基 NaOH の酸性塩）
　→ 塩基性！（「強塩基＋弱酸の酸性塩」の場合のみ塩基性になる特殊ルール）
⚠ NaHCO₃（重曹）は水溶液が塩基性 ── これがテストの罠！
【Step4：弱酸・弱塩基の遊離】
強酸 + 弱酸の塩 → 弱酸が遊離
ⓘ
「強い方が勝つ」の応用：弱酸の塩に強酸を加えると、弱酸が追い出される。
例①：CH₃COONa + HCl → CH₃COOH + NaCl（酢酸が遊離）
例②：Na₂CO₃ + 2HCl → 2NaCl + H₂O + CO₂↑
例③：CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑（石灰石＋塩酸）
強塩基 + 弱塩基の塩 → 弱塩基が遊離
ⓘ
例：NH₄Cl + NaOH → NaCl + H₂O + NH₃↑（アンモニアが発生）
🧪 実験室でのアンモニア発生法：塩化アンモニウム＋水酸化カルシウム を加熱！
2NH₄Cl + Ca(OH)₂ → CaCl₂ + 2H₂O + 2NH₃↑

重要事項⑤ 〜中和反応の計算〜

「H⁺の物質量 ＝ OH⁻の物質量」で解ける！▶
【Step1：中和の基本の式】
これ1本ですべて解ける
ⓘ
過不足なく中和するには、
「酸が出す H⁺ の物質量」＝「塩基が出す OH⁻ の物質量」

★中和の基本式★
a × c × V = b × c' × V'

a：酸の価数　c：酸のモル濃度[mol/L]　V：酸の体積[L]
b：塩基の価数　c'：塩基のモル濃度　V'：塩基の体積
💡 電離度 α には依存しない！ 弱酸でも最終的にはすべて中和されるため、α は式に入らない。
【Step2：基本計算例】
1価と1価の中和
ⓘ
例題1：0.10 mol/L の NaOH 水溶液 20 mL を中和するのに、濃度未知の HCl 15 mL 必要だった。HCl のモル濃度は？

解答：1 × c × (15/1000) = 1 × 0.10 × (20/1000)
　　c = 0.10 × 20 ÷ 15 = 約 0.13 mol/L
2価×1価
ⓘ
例題2：0.050 mol/L の H₂SO₄ 10 mL を中和するのに必要な 0.10 mol/L の NaOH の体積は？

解答：2 × 0.050 × (10/1000) = 1 × 0.10 × (V/1000)
　　V = 10 mL
💡 H₂SO₄ は2価なので H⁺ が2倍出る → NaOH は同じ物質量では足りない！
【Step3：応用パターン】
質量→物質量に変換
ⓘ
例題3：NaOH（式量40）0.40g を水に溶かして100mLとした水溶液を、0.10 mol/L の HCl で中和するのに必要な体積は？

NaOHの物質量 = 0.40 ÷ 40 = 0.010 mol
NaOHの濃度 = 0.010 ÷ 0.10 = 0.10 mol/L
中和式：1×0.10×0.10 = 1×0.10×V → V = 0.10 L ＝ 100 mL
気体の体積（標準状態22.4L）
ⓘ
例題4：標準状態で 224 mL のアンモニア NH₃ をすべて吸収するのに必要な 0.10 mol/L の HCl は何mL？

NH₃ の物質量 = 224 ÷ 22400 = 0.010 mol
NH₃は1価の塩基 → NH₃ + HCl → NH₄Cl
中和式：1×c×V(HCl) = 1×0.010
0.10 × V = 0.010 → V = 0.10 L ＝ 100 mL
気体分析に使う
ⓘ
逆滴定＝過剰の既知濃度の酸（塩基）を加え、余った分を別の塩基（酸）で滴定して求める方法。
例題5：発生したNH₃を 0.10 mol/L の HCl 20 mL にすべて吸収させた。残った HCl を 0.10 mol/L の NaOH 12 mL で中和した。NH₃の物質量は？

❶ 加えた HCl（H⁺）：0.10 × 0.020 = 2.0×10⁻³ mol
❷ NaOH（OH⁻）で中和された余りの HCl：0.10 × 0.012 = 1.2×10⁻³ mol
❸ NH₃ が吸収した H⁺ の物質量 = ❶ - ❷ = 0.8×10⁻³ mol

超頻出！

器具用途共洗い理由
メスフラスコ正確な濃度の溶液をつくる不要（純水で濡れたままOK）あとで純水を標線まで加えて希釈するため、水滴が残っても濃度に影響しない

メスフラスコ・コニカルビーカーは「ト」がつかない → 共洗い不要！
補助器具
ⓘ
駒込ピペット：おおよその体積を測って移すのに使用（メスフラスコに純水を加える最後の微調整）。

❶ メスフラスコに酸（塩基）を入れ、標線まで純水を加えて一定濃度の溶液を調製

重要事項⑦ 〜滴定曲線と二段階滴定〜

曲線の形と指示薬を完全マスター！▶
【Step1：滴定曲線とは】
縦軸pH・横軸滴下量
ⓘ
滴定曲線＝滴下量（横軸）と pH（縦軸）の関係を表すグラフ。
中和点付近で pH が急激に変化する（垂直に伸びる）のが特徴。
🎯 中和点で急激な pH ジャンプが起きる範囲を「pHジャンプ」とよぶ。
このジャンプ範囲に変色域が入っている指示薬を選ぶのが鉄則！
【Step2：4つの滴定曲線パターン】
中和点 pH ＝ 7（両方の指示薬OK）
ⓘ
例：HCl ＋ NaOH
中和点 pH ＝ 7（中性）
pHジャンプの範囲がとても広い（約 pH3〜pH11）
→ メチルオレンジ MO・フェノールフタレイン PP どちらでもOK
　pH 14│　　　　　　　　　　┌─ 　 │　　　　　　　　　／ 　7│─ ─ ─ ─ ─ ─ ─┤ ←ジャンプ（急激な変化） 　 │　　　　　　　　／ 　0│───────┘ 　　　0　　　　　　中和点　　　塩基滴下量
中和点 pH ＞ 7（PP使用）
ⓘ
例：CH₃COOH ＋ NaOH
中和点 pH ＞ 7（塩基性）（生じる CH₃COONa が加水分解して塩基性）
pHジャンプは塩基性側にのみ現れる（pH7〜pH11 付近）
→ フェノールフタレイン PP を使用（変色域 pH 8〜10 が範囲内）
→ MO は使えない（変色域 pH 3〜5 でジャンプ前に色が変わる）
中和点 pH ＜ 7（MO使用）
ⓘ
例：HCl ＋ NH₃
中和点 pH ＜ 7（酸性）（生じる NH₄Cl が加水分解して酸性）
pHジャンプは酸性側にのみ現れる（pH3〜pH7付近）
→ メチルオレンジ MO を使用（変色域 pH 3〜5 が範囲内）
→ PP は使えない
中和点 pH ≒ 7だが滴定不可
ⓘ
例：CH₃COOH ＋ NH₃
中和点 pH ≒ 7だが、pHジャンプがほぼ現れない！
→ どの指示薬も使えない → 滴定に不向き
これだけ暗記！
ⓘ
組み合わせ中和点 pH使える指示薬
強酸 + 強塩基= 7MO・PP 両方OK
弱酸 + 強塩基> 7PP のみ（強塩基側で変色）
強酸 + 弱塩基< 7MO のみ（強酸側で変色）
弱酸 + 弱塩基≒ 7どちらも使えない
💡 覚え方：「強」がついている側で pH がジャンプする → その側で色が変わる指示薬を選ぶ！
（強酸→酸性側で変わる MO／強塩基→塩基性側で変わる PP）
【Step3：二段階滴定（Na₂CO₃ の滴定）】
中和が2段階で進む
ⓘ
炭酸ナトリウム Na₂CO₃ は2価の弱塩基で、HCl で中和すると2段階に反応が進む。
【第1段階（第1中和点まで）】塩基 → 弱塩基へ
Na₂CO₃ + HCl → NaCl + NaHCO₃　（強い塩基性 → 弱い塩基性）
　→ 中和点 pH ＞ 7 → 指示薬 PP（フェノールフタレイン）で赤 → 無色

【第2段階（第1→第2中和点）】弱塩基 → 弱酸へ
NaHCO₃ + HCl → NaCl + H₂O + CO₂↑　（弱い塩基性 → 酸性）
　→ 中和点 pH ＜ 7 → 指示薬 MO（メチルオレンジ）で黄 → 赤
　pH 　　│　　　　│　　　　│ 　11│─┐　　│　　　　│ 　　│　└─┐（第1中和点／PP変色） 　 7│─ ─│─┐　　│ 　　│　　│　└─┐（第2中和点／MO変色） 　 3│　　│　　│└── 　　└──┴──┴──── 滴下量 　　　　　v1　　v2
頻出パターン！
ⓘ
NaOH（xmol）と Na₂CO₃（ymol）の混合水溶液を HCl で滴定するとき：
【第1段階】強塩基 → 弱塩基へ（PP：赤→無）
❶ NaOH + HCl → NaCl + H₂O 　　　　（x mol の HCl 消費）
❷ Na₂CO₃ + HCl → NaCl + NaHCO₃　（y mol の HCl 消費）
→ 第1中和点までの HCl 消費量：(x + y) mol

【第2段階】弱塩基 → 弱酸へ（MO：黄→赤）
❸ NaHCO₃ + HCl → NaCl + H₂O + CO₂　（y mol の HCl 消費）
→ 第1→第2中和点までの HCl 消費量：y mol
💡 x, y の求め方
① 第1中和点までの滴下量 V₁[L] × 濃度 c[mol/L] = x + y
② 第1→第2中和点までの滴下量 V₂[L] × 濃度 c = y
③ 差し引いて x = (V₁-V₂)×c、y = V₂×c を求める
共通テスト頻出
ⓘ
例題：NaOH と Na₂CO₃ の混合水溶液に 0.10 mol/L HCl を滴下したところ、PP の色が消えたのは 25 mL、MO の色が変わったのはさらに 10 mL 追加した時点だった。含まれる NaOH と Na₂CO₃ の物質量は？

❶ 第1→第2中和点分：y mol = 0.10 × (10/1000) = 1.0×10⁻³ mol（Na₂CO₃）
❷ 第1中和点まで：x + y mol = 0.10 × (25/1000) = 2.5×10⁻³ mol
❸ よって NaOH = x = 2.5×10⁻³ - 1.0×10⁻³ = 1.5×10⁻³ mol

答え：NaOH = 1.5×10⁻³ mol、Na₂CO₃ = 1.0×10⁻³ mol
第2部 物質の変化 ／ 3、酸化還元反応

重要事項① 〜酸化還元反応とは何か（酸化数）〜

「電子のやり取り」で見抜く！▶
【Step1：酸化・還元の3つの見方】
酸素/水素/電子/酸化数
ⓘ
酸化とは、次のどれかが起こること：
❶ 酸素O を受け取る　（例：C + O₂ → CO₂）
❷ 水素H を失う　（例：H₂S → S）
❸ 電子e⁻ を失う　（例：Zn → Zn²⁺ + 2e⁻）
❹ 酸化数が増加する　（例：Fe²⁺ → Fe³⁺、+2→+3で増加）
💡 現代的定義では 「電子を失うこと＝酸化」 が最も本質的。
酸化の逆
ⓘ
還元とは、次のどれかが起こること（酸化の完全な逆）：
❶ 酸素O を失う　（例：CuO → Cu）
❷ 水素H を受け取る　（例：N₂ → NH₃）
❸ 電子e⁻ を受け取る　（例：Cu²⁺ + 2e⁻ → Cu）
❹ 酸化数が減少する
🔑 酸化と還元は必ず同時に起こる！ 電子を出す物質があれば、必ず受け取る物質があるため。→ 酸化還元反応とセットで呼ぶ。
【Step2：酸化数の決定ルール】
4つの原則を覚える
ⓘ
❶ 単体：すべての原子の酸化数は0
　例：H₂、O₂、Cu、Fe、Zn　→ すべて 0

❷ 単原子イオン：酸化数はイオンの価数（電荷）に等しい
　例：Na⁺=+1、Cl⁻=-1、Fe²⁺=+2、Al³⁺=+3

❸ 化合物中：すべての原子の酸化数の和 = 0
　例：H₂O → +1×2 + (-2) = 0 ✓

❹ 多原子イオン：すべての原子の酸化数の和 = イオンの価数
　例：SO₄²⁻ → S + (-2)×4 = -2 ∴ S = +6
優先順位と例外
ⓘ
優先元素原則例外
1アルカリ金属（1族）+1なし
2アルカリ土類金属（2族）+2なし
3フッ素 F-1なし
4水素 H+1金属水素化物 NaH、CaH₂ では -1
5酸素 O-2過酸化物 H₂O₂・Na₂O₂ では -1／OF₂ では +2
💡 優先順位が高い元素から先に決めて、残った元素は「化合物の和=0」から逆算する！
頻出化合物
ⓘ
物質着目原子酸化数計算
KMnO₄Mn+7+1 + Mn + (-2)×4 = 0
K₂Cr₂O₇Cr+6+1×2 + Cr×2 + (-2)×7 = 0
H₂SO₄S+6+1×2 + S + (-2)×4 = 0
HNO₃N+5+1 + N + (-2)×3 = 0
NH₃N-3N + (+1)×3 = 0
H₂O₂O-1（過酸化物）+1×2 + O×2 = 0
NaHH-1（例外）+1 + H = 0
【Step3：酸化還元の判定】
酸化数の増減をチェック
ⓘ
手順：反応前後の各原子の酸化数を求め、変化があれば酸化還元反応！
例①：2Cu + O₂ → 2CuO
　Cu：0 → +2（酸化）／O：0 → -2（還元）
　→ 酸化還元反応（酸化と還元が同時に起きている）

例②：HCl + NaOH → NaCl + H₂O
　H：+1→+1／Cl：-1→-1／Na：+1→+1／O：-2→-2　変化なし
　→ 中和反応（酸化還元ではない）
💡 中和反応・沈殿反応は基本的に「電子の授受なし」→ 酸化還元ではない！
「自分」と「相手」の視点
ⓘ
酸化剤＝相手を酸化する（＝自分は還元される／電子を受け取る）
還元剤＝相手を還元する（＝自分は酸化される／電子を失う）
⚠ 「酸化剤」は自分が「酸化される」わけではない！
「酸化剤」は"相手を酸化する薬剤"のことなので、自分自身は還元されている。用語のトラップに注意！

この順に組み立てる！
ⓘ
【手順】
❶ 酸化剤・還元剤を特定し、「反応前 → 反応後」を書く
❷ O、H 以外の原子の数を合わせる
❸ 不足しているO を H₂O で合わせる
❹ 不足しているH を H⁺ で合わせる
❺ 両辺の電荷（±）が等しくなるよう e⁻ を加える

❶ MnO₄⁻ → Mn²⁺（反応前後を書く）
❷ Mn の数は左右1個ずつでOK
❸ 左辺に O が4個、右辺に0個 → 右辺に H₂O を4つ
　　MnO₄⁻ → Mn²⁺ + 4H₂O
❹ 右辺の H が 8個 → 左辺に H⁺ を 8個
　　MnO₄⁻ + 8H⁺ → Mn²⁺ + 4H₂O
❺ 電荷：左辺 = -1+8=+7、右辺 = +2 → 左辺が5多い → 左辺に e⁻ を5個
　　MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O　完成！
実演②：シュウ酸(COOH)₂の半反応式

❶ (COOH)₂ → 2CO₂
❷ C の数：左右 2 で OK
❸ O の数：左右 4 で OK
❹ 左辺の H が2個、右辺0個 → 右辺に H⁺ を2個
　　(COOH)₂ → 2CO₂ + 2H⁺
❺ 電荷：左0、右+2 → 右に e⁻ を2個
　　(COOH)₂ → 2CO₂ + 2H⁺ + 2e⁻　完成！

ⓘ
【手順】
❶ 酸化剤・還元剤それぞれの半反応式を作る
❷ e⁻ の数が等しくなるよう、両式を整数倍する
❸ 両式を足し合わせ、e⁻を消去する
❹ 必要ならスペクテーターイオン（陽イオンなど）を加えて「化学反応式」に整える
実演：KMnO₄ と (COOH)₂ の反応（硫酸酸性）

❶ MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O 　　… ①（酸化剤／e⁻5個受取）
❷ (COOH)₂ → 2CO₂ + 2H⁺ + 2e⁻ 　　… ②（還元剤／e⁻2個放出）

❸ e⁻の最小公倍数10：①×2 と ②×5
　　2MnO₄⁻ + 16H⁺ + 10e⁻ → 2Mn²⁺ + 8H₂O
　　5(COOH)₂ → 10CO₂ + 10H⁺ + 10e⁻

❹ 足し合わせて e⁻ を消す：
　　2MnO₄⁻ + 16H⁺ + 5(COOH)₂ → 2Mn²⁺ + 8H₂O + 10CO₂ + 10H⁺
❺ H⁺ を整理（16-10=6）：
　　2MnO₄⁻ + 6H⁺ + 5(COOH)₂ → 2Mn²⁺ + 8H₂O + 10CO₂

これに K⁺、SO₄²⁻ を加えて完全な化学反応式にすると：
2KMnO₄ + 3H₂SO₄ + 5(COOH)₂ → K₂SO₄ + 2MnSO₄ + 8H₂O + 10CO₂

重要事項③ 〜酸化剤と還元剤の量的関係（酸化還元滴定）〜

「電子の物質量」が等しい！▶
【Step1：量的関係の式】
e⁻の授受でつり合わせる
ⓘ
★ 酸化還元滴定の量的関係 ★

（酸化剤の物質量）×（受け取る e⁻ の数）
　　　　　　　　　　　　　　＝
（還元剤の物質量）×（放出する e⁻ の数）
💡 e⁻の物質量が等しい！という考え方。酸塩基滴定の「H⁺の物質量が等しい」と対応する。
【Step2：KMnO₄とシュウ酸の滴定】
指示薬いらず！
ⓘ
【手順】
❶ シュウ酸水溶液をコニカルビーカーに入れる
❷ 希H₂SO₄を加えて硫酸酸性にする
❸ 60〜80℃に加温する（反応速度を上げるため）
❹ 濃度未知のKMnO₄水溶液をビュレットから滴下
❺ 水溶液が薄い赤紫色に色付いた点を終点とする
💡 指示薬は不要！ KMnO₄自身が赤紫色なので、反応が終わった直後から色が消えなくなる。
→ KMnO₄自身が指示薬の役割を果たす。
HCl や HNO₃ は NG
ⓘ
KMnO₄ を酸性条件で使うが、希硫酸 H₂SO₄を使う理由：
❌ HCl は NG：Cl⁻ が KMnO₄ に酸化されて Cl₂ になってしまう
　　（KMnO₄ の Mn²⁺ 生成分が Cl⁻ の酸化に消費される → 誤差）

❌ HNO₃ は NG：HNO₃ 自身が酸化剤として働いてしまい、シュウ酸を酸化する
　　→ KMnO₄との滴定が正確にできない

✅ H₂SO₄ は OK：SO₄²⁻ は酸化されにくく、H₂SO₄ 自身も酸化剤として働かない（希硫酸なので）
KMnO₄×5 = シュウ酸×2
ⓘ
例題：硫酸酸性のもと、0.050 mol/L のシュウ酸(COOH)₂水溶液 8.0 mL に、濃度未知のKMnO₄水溶液 10 mL を加えたら過不足なく反応した。KMnO₄のモル濃度は？

❶ KMnO₄ は e⁻ を 5 個受け取る（MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O）
❷ (COOH)₂ は e⁻ を 2 個放出する（(COOH)₂ → 2CO₂ + 2H⁺ + 2e⁻）

量的関係式：
c × (10/1000) × 5 = 0.050 × (8.0/1000) × 2
c × 0.010 × 5 = 0.050 × 0.008 × 2
c = 0.00080 ÷ 0.050 = 0.016 mol/L
【Step3：ヨウ素滴定】
2段階の滴定・デンプン指示薬
ⓘ
目的：濃度未知の酸化剤（例：H₂O₂）を、Na₂S₂O₃（チオ硫酸ナトリウム）で間接的に滴定する。
【第1段階】試料の酸化剤（H₂O₂）に過剰のKIを加える
　H₂O₂ + 2KI + H₂SO₄ → I₂ + 2H₂O + K₂SO₄
　→ ヨウ素 I₂ が遊離（褐色）

【第2段階】生じた I₂ を Na₂S₂O₃ で滴定
　I₂ + 2Na₂S₂O₃ → 2NaI + Na₂S₄O₆
　（テトラチオン酸ナトリウム）
🎨 指示薬：デンプン水溶液
・I₂ が残っている間はデンプンと結合して青紫色を示す（ヨウ素デンプン反応）
・I₂ が I⁻ に変わりきると青紫色が消える → これが終点！
【Step4：COD（化学的酸素要求量）】
水質汚濁の指標
ⓘ
COD（Chemical Oxygen Demand）＝家庭排水などに含まれる有機物によって、湖沼や海水がどの程度汚染されているかを示す指標。
有機物を酸化剤（KMnO₄）で酸化するのに必要な酸化剤の量を、O₂ の質量（mg/L）に換算した値。
【測定手順】
❶ 試料水に H₂SO₄ と過剰のKMnO₄を加え、加熱して有機物を分解
❷ 残った KMnO₄ を過剰のNa₂C₂O₄（シュウ酸ナトリウム）で還元
❸ 残ったシュウ酸ナトリウムをKMnO₄で逆滴定

関係式：
（KMnO₄が受け取ったe⁻ 総量）＝（有機物が失ったe⁻）＋（Na₂C₂O₄が失ったe⁻）

最終的に O₂ の質量に換算：O₂ + 4H⁺ + 4e⁻ → 2H₂O から、O₂ 1mol = e⁻ 4mol

重要事項④ 〜酸化剤・還元剤としての強さ〜

強い方が「電子を欲しがる」！▶
【Step1：ハロゲンの酸化力】
F₂ > Cl₂ > Br₂ > I₂
ⓘ
ハロゲン単体の酸化力（＝e⁻を受け取る力の強さ）：
F₂ ＞ Cl₂ ＞ Br₂ ＞ I₂

（周期表で上ほど酸化力が強い＝原子半径が小さく電子を引き寄せやすい）
強い酸化剤が電子を奪う
ⓘ
「酸化力の強い単体」が「酸化力の弱い単体のイオン」から電子を奪って反応する。
例①：2NaBr + Cl₂ → 2NaCl + Br₂　（反応する）
　　→ Cl₂ は Br₂ より酸化力が強い → 反応進行 ✓

例②：2NaCl + Br₂ → 反応しない
　　→ Br₂ は Cl₂ より酸化力が弱い → 反応進まない ✗

例③：2KI + Br₂ → 2KBr + I₂ （反応する）
　　→ 溶液が褐色に着色（I₂ が生成）
💡 ヨウ素デンプン反応：I₂ が生じたことは、デンプン水溶液に加えて青紫色になれば確認できる！
【Step2：H₂O₂ と SO₂ の両性】
頻出パターン
ⓘ
❶ H₂O₂（過酸化水素）
・基本は酸化剤：H₂O₂ + 2H⁺ + 2e⁻ → 2H₂O
・相手がMnO₄⁻（もっと強い酸化剤）のとき → 還元剤となり O₂ を発生
　H₂O₂ → O₂ + 2H⁺ + 2e⁻

❷ SO₂（二酸化硫黄）
・基本は還元剤：SO₂ + 2H₂O → SO₄²⁻ + 4H⁺ + 2e⁻
・相手がH₂S（もっと強い還元剤）のとき → 酸化剤となり S を析出
　SO₂ + 4H⁺ + 4e⁻ → S + 2H₂O
　（SO₂ + 2H₂S → 3S + 2H₂O：温泉地の硫黄析出）
【Step3：金属の酸化力・還元力】
イオン化傾向大＝還元力大
ⓘ
イオン化傾向が大きい金属ほど、還元剤として強い（e⁻を放出しやすい）
イオン化傾向が小さい金属イオンほど、酸化剤として強い（e⁻を受け取りやすい）
例：CuSO₄水溶液に Zn を入れると、Cu が析出する
Zn + Cu²⁺ → Zn²⁺ + Cu
（Zn は Cu より還元力が強い → Cu²⁺ から電子を奪える）
→ 詳細は次項「重要事項⑤ イオン化傾向」で扱う！

重要事項⑤ 〜金属のイオン化傾向と反応〜

「リッチに貸そかな…」の完全マスター！▶
【Step1：イオン化列と語呂合わせ】
Li〜Au、17元素
ⓘ
イオン化傾向＝金属が水溶液中で陽イオンになろうとする傾向。
大 ← Li ＞ K ＞ Ca ＞ Na ＞ Mg ＞ Al ＞ Zn ＞ Fe ＞ Ni ＞ Sn ＞ Pb ＞ (H₂) ＞ Cu ＞ Hg ＞ Ag ＞ Pt ＞ Au → 小
🎵 語呂合わせ：「リッチに 貸そ か な！ ま あ あ て に す な！ ひ ど す ぎ 借 金！」

　リッチに　＝ Li
　貸そ　　＝ K
　か　　　＝ Ca
　な　　　＝ Na
　ま　　　＝ Mg
　あ　　　＝ Al
　あ　　　＝ Zn（あえん）
　て　　　＝ Fe（てつ）
　に　　　＝ Ni
　す　　　＝ Sn（すず）
　な　　　＝ Pb（なまり）
　ひ　　　＝ (H₂)
　ど　　　＝ Cu（どう）
　す　　　＝ Hg（水銀＝すいぎん）
　ぎ　　　＝ Ag（銀）
　借　　　＝ Pt（白金）
　金　　　＝ Au
【Step2：水との反応】
常温水／熱水／高温水蒸気
ⓘ
反応該当金属反応式（例）
常温の水と反応Li, K, Ca, Na2Na + 2H₂O → 2NaOH + H₂↑
熱水と反応MgMg + 2H₂O → Mg(OH)₂ + H₂↑
高温の水蒸気と反応Al, Zn, Fe3Fe + 4H₂O → Fe₃O₄ + 4H₂↑
反応しないNi以降（Ni, Sn, Pb, Cu, Hg, Ag, Pt, Au）―
💡 覚え方：「常温はリッチに貸そかなまで、熱水はマグ、水蒸気はあ・あ・て（Al, Zn, Fe）まで！」
【Step3：酸との反応】
H₂より上の金属
ⓘ
希塩酸・希硫酸と反応してH₂ を発生するのは、H₂よりイオン化傾向が大きい金属（Li〜Pb まで）。
例①：Zn + 2HCl → ZnCl₂ + H₂↑
例②：Fe + H₂SO₄ → FeSO₄ + H₂↑
⚠ Pb は表面がPbCl₂やPbSO₄で被覆されて反応が止まるため、実質的にはあまり進まない（テストの落とし穴）
Cu, Hg, Ag も溶ける
ⓘ
H₂より下の Cu、Hg、Ag は希酸には溶けないが、酸化力の強い酸には溶ける！
❶ 銅と希硝酸
3Cu + 8HNO₃ → 3Cu(NO₃)₂ + 4H₂O + 2NO↑（無色気体）

❷ 銅と濃硝酸
Cu + 4HNO₃ → Cu(NO₃)₂ + 2H₂O + 2NO₂↑（赤褐色気体）

❸ 銅と熱濃硫酸
Cu + 2H₂SO₄ → CuSO₄ + 2H₂O + SO₂↑（刺激臭気体）
💡 これらの酸は電子を受け取る作用（酸化力）によって、H₂を発生させない代わりに金属を溶かす。
H⁺ ではなく NO₃⁻ や SO₄²⁻ が酸化剤として働く！
濃硝酸に溶けない
ⓘ
不動態＝金属表面に緻密な酸化被膜ができて、内部への反応が進まなくなる状態。
不動態を作る金属：Fe, Co, Ni, Al, Cr（濃硝酸には溶けない！）

🎵 語呂合わせ：「鉄子にある苦労！」
　鉄（Fe）子（Co）に（Ni）ある（Al）苦労（Cr）
💡 アルミ製の容器で濃硝酸を運搬できるのはこの性質のおかげ！
Pt, Au も溶かす
ⓘ
王水＝濃硝酸 : 濃塩酸 ＝ 1 : 3（体積比）で混合した溶液。
最強の酸化力を持ち、Pt（白金）や Au（金）も溶かす！
🎵 語呂合わせ：「一生は三円！」
　一（濃硝酸1）生（硝酸の「しょう」）は（体積比）三（濃塩酸3）円（塩酸の「えん」）
例：Au + HNO₃ + 4HCl → HAuCl₄ + NO + 2H₂O（テトラクロロ金(III)酸）
【Step4：空気との反応】
常温／加熱／強熱／反応しない
ⓘ
反応該当金属例
常温で速やかに酸化Li, K, Ca, Na4Na + O₂ → 2Na₂O
加熱で酸化Mg, Al, Zn, Fe, Ni, Sn, Pb, Cu2Cu + O₂ → 2CuO
強熱で酸化Hg, Ag2Hg + O₂ → 2HgO
反応しないPt, Au―（永遠に光沢を保つ！）
💡 だから金や白金はジュエリーとして人気！ 銀は強熱でも黒く硫化しやすいので手入れが必要。
【Step5：★超重要★ イオン化傾向と反応性の総まとめ表】
これだけで完璧！
ⓘ
イオン化傾向大　←──────────────────────→　小
元素Li, K, Ca, NaMgAl, Zn, FeNi, Sn, Pb, (H₂), Cu, Hg, Ag, Pt, Au
水との反応常温の水と反応熱水と反応高温水蒸気と反応反応しない
希酸との反応希塩酸・希硫酸と反応してH₂発生（Ni, Sn, Pbまで含む）Cu, Hg, Ag → 酸化力の強い酸のみ／Pt, Au → 王水のみ
空気との反応常温で速やかに酸化加熱で酸化Hg, Ag → 強熱で／Pt, Au → 反応しない
製法溶融塩電解（融解した塩を電気分解）CO 還元・電解精錬など
★ 覚え方総まとめ ★

酸：「H₂までは希酸OK、Cu Hg Agは酸化力強い酸、Pt Auは王水のみ！」
空気：「リッチに貸そかな速やかに、次は加熱、Hg Agは強熱、Pt Auは反応しない」

重要事項⑥ 〜電池（一次・二次・燃料）〜

酸化還元反応 → 電気エネルギーへ！▶
【Step1：電池の基本原理】
正極・負極・起電力
ⓘ
電池＝酸化還元反応の化学エネルギーを電気エネルギーに変換する装置。
負極（-極）＝還元剤が e⁻ を放出する電極（＝酸化される極）
　　　　　　　　→ イオン化傾向の大きい金属

正極（+極）＝酸化剤が e⁻ を受け取る電極（＝還元される極）
　　　　　　　　→ イオン化傾向の小さい金属

起電力＝両極間の電位差（電池のパワー）
　　　　　　→ 2極のイオン化傾向の差が大きいほど大きい

放電＝電池から電流を取り出すこと
充電＝外部電源で逆反応を起こし、電池を元に戻すこと
💡 電子は 負極 → 導線 → 正極 の順で流れる。電流はその逆方向（+ → -）に流れる。
充電できるか否か
ⓘ
分類特徴代表例
一次電池充電できない（使い切り）マンガン乾電池／アルカリマンガン乾電池／酸化銀電池／リチウム電池
二次電池（蓄電池）充電して繰り返し使える鉛蓄電池／リチウムイオン電池／ニッケル水素電池
燃料電池燃料と酸素を供給し続けて発電（連続型）リン酸形／固体高分子形
【Step2：基礎電池：ボルタ／ダニエル】
歴史上初の電池・分極問題あり
ⓘ
構成：(-) Zn ｜ H₂SO₄ aq ｜ Cu (+)
起電力：約 0.76 V（一次電池）

負極反応（Zn 板が溶ける）：
　Zn → Zn²⁺ + 2e⁻

正極反応（Cu 板の表面で H₂ が発生）：
　2H⁺ + 2e⁻ → H₂↑

全体：Zn + 2H⁺ → Zn²⁺ + H₂
⚠ 分極：正極表面にH₂が付着 → 電流が流れにくくなる
→ この問題を解消したのがダニエル電池。
素焼き板で分極を防ぐ
ⓘ
構成：(-) Zn ｜ ZnSO₄ aq ｜｜ CuSO₄ aq ｜ Cu (+)
（｜｜ は素焼き板を表す）
起電力：約 1.10 V（一次電池）

負極反応：Zn → Zn²⁺ + 2e⁻（Zn板が溶ける）
正極反応：Cu²⁺ + 2e⁻ → Cu（Cu板にCuが析出）

全体：Zn + Cu²⁺ → Zn²⁺ + Cu
🧪【ダニエル電池の図】
　［Zn板］—ZnSO₄水溶液｜素焼き板｜CuSO₄水溶液—［Cu板］
　　（-極）　　　　　　　　　　　　　　　　　　（+極）
💡 素焼き板の役割：ZnSO₄とCuSO₄が急激に混ざるのを防ぐ
（もし混ざると、Zn板が直接Cu²⁺と反応して電子が導線を流れず、電池として機能しない）

💡 ダニエル電池を長持ちさせるコツ：
❶ ZnSO₄ の濃度は小さく（Zn²⁺が溶けやすい環境）
❷ CuSO₄ の濃度は大きく（Cu が析出しやすい環境）
【Step3：実用電池 一次／二次】
自動車バッテリー
ⓘ
構成：(-) Pb ｜ H₂SO₄ aq ｜ PbO₂ (+)
起電力：約 2.00 V

【放電時】
負極：Pb + SO₄²⁻ → PbSO₄ + 2e⁻
正極：PbO₂ + 4H⁺ + SO₄²⁻ + 2e⁻ → PbSO₄ + 2H₂O
全体：Pb + PbO₂ + 2H₂SO₄ → 2PbSO₄ + 2H₂O

【充電時】：放電の逆反応が起きる（電気分解！）
負極：PbSO₄ + 2e⁻ → Pb + SO₄²⁻
正極：PbSO₄ + 2H₂O → PbO₂ + 4H⁺ + SO₄²⁻ + 2e⁻
💡 放電すると硫酸が薄くなる（H₂SO₄が消費、H₂Oが生成）
→ 電解液の密度を測れば充電状態がわかる！
自動車バッテリーの点検はこの原理を利用。
H₂ + O₂ → H₂O のみ
ⓘ
燃料電池＝水素などの燃料と酸素を供給して発電する装置。
❶ リン酸形燃料電池（電解液：H₃PO₄）
負極：H₂ → 2H⁺ + 2e⁻（H₂ が電子を放出＝還元剤）
正極：O₂ + 4H⁺ + 4e⁻ → 2H₂O（O₂ が電子を受け取る＝酸化剤）
全体：2H₂ + O₂ → 2H₂O

❷ 固体高分子形燃料電池（電解液：KOH aqなどアルカリ）
負極：H₂ + 2OH⁻ → 2H₂O + 2e⁻
正極：O₂ + 2H₂O + 4e⁻ → 4OH⁻
全体：2H₂ + O₂ → 2H₂O
🌱 発生する物質が水だけ＝クリーンエネルギー！
FCV（燃料電池自動車）や家庭用エネファームで実用化。
現代のスマホ・EV

用途：スマートフォン、ノートPC、電気自動車（EV）

重要事項⑦ 〜工業的製法（Al・Fe・Cu）〜

酸化還元を利用して金属を作る！▶
【Step1：アルミニウムの製法（溶融塩電解）】
Alはイオン化傾向が大きすぎる
ⓘ
Al はイオン化傾向が非常に大きく、水溶液の電気分解では H₂O が先に還元されてしまうため、Alを得ることができない！
→ アルミナ Al₂O₃ を融解して電気分解する（溶融塩電解）。
⚠ Al₂O₃ の融点は約 2054℃と非常に高い！　これを下げる工夫が必要。
4段階の製造工程
ⓘ
Step1：ボーキサイトを濃NaOH水溶液に溶かす
　Al₂O₃ + 2NaOH + 3H₂O → 2Na[Al(OH)₄]（テトラヒドロキシドアルミン酸ナトリウム）

Step2：多量の水で希釈し、Al(OH)₃を沈殿させる
　Na[Al(OH)₄] → Al(OH)₃↓ + NaOH

Step3：Al(OH)₃を加熱してアルミナ Al₂O₃ を得る
　2Al(OH)₃ → Al₂O₃ + 3H₂O

Step4：氷晶石 Na₃AlF₆ に Al₂O₃ を溶かして溶融塩電解
　陰極：Al³⁺ + 3e⁻ → Al（アルミニウムが析出）
　陽極：C + O²⁻ → CO + 2e⁻　および　C + 2O²⁻ → CO₂ + 4e⁻
　　　（陽極の炭素電極が O²⁻ と反応して消費される）
💡 氷晶石 Na₃AlF₆ の役割：Al₂O₃（融点2054℃）の融点を下げて、約 1000℃ でも融解させる融剤。
→ これにより電気エネルギーの消費を大幅削減。
⚡ Al 1kg 製造に電力 約14kWh が必要！ アルミは「電気の缶詰」とも呼ばれる。
→ リサイクルすれば新規製造の1/30以下のエネルギーで済む。
【Step2：鉄の製法（溶鉱炉）】
CO で酸化鉄を還元
ⓘ
原料：鉄鉱石（Fe₂O₃、Fe₃O₄）、コークス C、石灰石 CaCO₃
❶ コークスの燃焼で CO を発生：
　　2C + O₂ → 2CO

❷ CO による還元で銑鉄を得る：
　　Fe₂O₃ + 3CO → 2Fe + 3CO₂

❸ 石灰石は鉄鉱石中の SiO₂ をスラグとして分離：
　　CaCO₃ → CaO + CO₂
　　CaO + SiO₂ → CaSiO₃（スラグ ＝ ケイ酸カルシウム）
炭素成分を減らす
ⓘ
銑鉄（せんてつ）：溶鉱炉から得られる、炭素を約 4% 含む鉄（もろい・鋳物用）

転炉での処理：銑鉄にO₂を吹き込み加熱 → 炭素を CO₂ として除去
　　C + O₂ → CO₂↑

鋼（こう）：炭素を約 0.02〜2% に減らした鉄（強靭・建築・レール用）
🏭【製鉄プロセス】
　鉄鉱石＋コークス＋石灰石
　　　　　↓（溶鉱炉：CO還元）
　　　銑鉄（C 約4%）
　　　　　↓（転炉：O₂吹込み）
　　　　鋼（C 少ない）
【Step3：銅の製法（電解精錬）】
粗銅 → 純銅（99.99%）
ⓘ
構成：陽極＝粗銅（99%程度）／陰極＝純銅（薄板）／電解液＝CuSO₄ aq（硫酸酸性）
【反応】
陽極（粗銅）：主に Cu → Cu²⁺ + 2e⁻（銅が溶け出す）
陰極（純銅）：Cu²⁺ + 2e⁻ → Cu（純銅が析出）
🔬【電解精錬の図】
　［粗銅］（+極）—CuSO₄水溶液—［純銅］（-極）
　　　　　　　　　　（電源＋直流電流）
　　　（Cu²⁺として溶ける）　　　　　（Cuとして析出）
金・銀が回収できる！
ⓘ
粗銅中の不純物のうち：
❶ Cu よりイオン化傾向が大きい（Zn, Fe, Ni など）
　→ Cuと一緒に Cu²⁺, Zn²⁺, Fe²⁺として電解液中に溶解（陰極では析出しない）

❷ Cu よりイオン化傾向が小さい（Ag, Au など）
　→ 電子を放出しないため溶けずに落下し、陽極の下に陽極泥（ようきょくでい）として沈殿
　→ この陽極泥から金や銀を回収できる！（副産物）

❸ Pb：PbSO₄ が難溶性なので陽極泥に沈殿する
💰 実は銅の電解精錬は「金・銀の回収工程」でもある！ 電気代のかなりの部分が回収された貴金属で相殺される。
純度を保つため
ⓘ
電流を小さくゆっくり流す理由：
・電流を大きくしすぎると、Cu より少し卑な不純物（Ni, Znなど）も陰極で析出してしまう
・ゆっくり電解すると、イオン化傾向の順で確実に銅だけが析出する
→ 純度 99.99% の純銅が得られる
【Step4：3つの製法の総まとめ】
イオン化傾向との対応
ⓘ
金属イオン化傾向製法主な反応
Al大（H₂より上）溶融塩電解（融解したAl₂O₃ + 氷晶石）陰極：Al³⁺ + 3e⁻ → Al
Fe中CO還元（溶鉱炉）Fe₂O₃ + 3CO → 2Fe + 3CO₂
Cu小（H₂より下）電解精錬（水溶液電気分解）陰極：Cu²⁺ + 2e⁻ → Cu
💡 イオン化傾向が大きい金属ほど、より強力な還元手段（電気分解）が必要！
・Al は水溶液の電気分解ではダメ → 溶融塩電解
・Fe は CO で還元できる（比較的簡単）
・Cu は水溶液の電気分解で高純度化できる
